ID Developmental Blog

Week 1

Introduction to Web Development

In this week, I learnt what the basics html and css. For html, the main things that I learnt was the different headings, and some other basic tags like img, list tags, link tags and container tags

Week 2

Semantic HTML and CSS 101

In this week, I learnt some HTML5 semantics and what is it's purpose, which is its positive impact on accessibility, especially for users who rely on assistive technologies to use the web. For CSS, I learnt how I can make use of things like developer mode to control css without making changes to the actual css file and to see roughly how the website would look like on other devices, like mobile. I also learnt different ways to change the styles of elemets, such as inline, style elements and external stylesheets. IDs and Classes were also something important from this week as it allows me to target elements. I also learnt how to use google fonts in my websites and also and the differences between padding and margin.

Week 3

More CSS, Media queries, Flex and Adobe XD

In this week, I learnt the different types of inputs for forms in HTML, and how to add things like labels and validations. I also learnt how to group elements in CSS and how inheritance works. CSS is works in a cascading manner so styles applied later will override previous styles to that element. !important overrides order however. I learnt how to make use of media queries to make the website responsive and look good on all devices, with mobile-first or web-first approaches. I also learnt the different positionings of elements and how they work, alongside the z-index and how it affects elements. I also learnt how to use display flex to make responsive websites and how to control flex items. The last thing is how to use AdobeXD to make wireframes to assist web development.

Week 4

Putting it all together

In this week, I learnt how to create responsive images, how to make use of GIT pushes and pulls and the basics of wireframing.

Week 5

Javascript essential training

In this week, I learnt basics of javascript, some things including how to make use of objects, classes and variables. I learnt the difference between methods and functions, that methods are inside the class while functions are outside. I also learnt how to use querySelector and getElementsBy to target elements.

Week 6

JS basics recap, Objects, DOM

In this week, I got a recap of JS basics like datatypes, operators, loops, arrays and functions. I also learnt how to define, and access objects in JS, like how to update them. I also learnt the differences between let, var and const in variable declaration. I also learnt how to access elements through DOM traversal method.

Week 7

DOM, event handling, validation, local storage

In this week, I got a recap on DOM, and learnt about event handling, like what are the different kinds of events eg. "mousemove" or "click" and how to use event listeners to act on that event. I also learnt JSON data is made up of key-value pairs and that local storage allows us to store up to 5mb of data on our browsers, and how to clear it, or get items from the local storage. I also learnt about session storage.

Week 8

jQuery, API

In this week, I learnt what jQuery is and how I can use it to make writing JS code easier. I also learnt about the use of AJAX and how it can be used to load data into a part of a page without having to refresh the entire page. I also learnt about APIs and how it takes requests, delivers it and brings back responses, and how to access APIs.

Week 9

Bootstrap

In this week, I learnt about bootstrap which is a frontend toolkit that makes web development much easier. I learnt that bootstrap uses flexbox grid for its layout, uses rem for mesurements and has device breakpoints to accommodate for different media sizes. It can be used to easily create things like a navbar, forms or carousels, that may take a while to create from scratch.

Week 12

RestDB and NoSQL

In this week, I learnt about using databases to store and retrieve data through RestDB. I learnt about the different requests such as GET, POST, PUT, and DELETE. I also learnt the difference between SQL and NoSQL.

Week 13

Best Practices and common mistakes testing

In this week, I learnt the common mistakes we made in ASG1 and good practices for web development and pushing to github. We should give proper commit messages that describe our changes well and not things like "made change".

Week 14

Lottie

In this week, I learnt about Lottie and how I can inplement lottie animations into my websites to make the website feel responsive, even when doing tasks like loading in information on a page or clicking on buttons that POST content. I also learnt how to customise and set timeouts for the animaitons.
